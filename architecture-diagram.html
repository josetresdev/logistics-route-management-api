Componentes Principales

  • Frontend: Angular 21.2 con Material Design y componentes standalone
  • Backend: Django REST Framework 3.14 con arquitectura DDD
  • Base de Datos: PostgreSQL 16 con esquema relacional completo
  • API Layer: ViewSets REST con autenticación token-based JWT
  • Business Logic: Servicios, validadores y casos de uso
  • Infrastructure: Repositorios y acceso a datos con Django ORM
  • DevOps: Docker Compose para orquestación de contenedores
