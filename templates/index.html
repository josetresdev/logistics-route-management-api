Logistics Route Management API

Sistema profesional para la gestión, validación y ejecución de rutas logísticas.
Esta aplicación permite administrar el ciclo completo de rutas: carga desde archivos Excel, validación de calidad de datos, almacenamiento persistente en PostgreSQL y exposición de APIs REST para operación, consulta y auditoría. Diseñada bajo principios de arquitectura limpia y separación de responsabilidades.

Instalación y ejecución con Docker
  • Clonar repositorio: git clone <repo-url>
  • Construir y ejecutar servicios: docker compose up --build
  • Ver logs en tiempo real: docker compose logs -f
  • Detener servicios: docker compose down
Endpoints principales
POST
/api/token-auth/
Obtener token de autenticación
Permite autenticarse y recibir un token para consumir la API.
GET
/api/routes/
Listar rutas
Devuelve todas las rutas registradas en el sistema.
POST
/api/routes/import
Importar rutas desde Excel
Carga masiva de rutas con validación automática de datos.
GET
/api/execution-logs/
Consultar registros de ejecución
Permite auditar ejecuciones y revisar resultados históricos.
