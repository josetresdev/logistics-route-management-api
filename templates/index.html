Logistics Route Management API

Sistema profesional para la gestión y optimización de rutas logísticas.
Esta prueba técnica consiste en construir un sistema web que permita gestionar el ciclo de vida de rutas logísticas desde su carga hasta su ejecución. El sistema debe recibir información en archivos Excel, validar la calidad de los datos, reportar errores, persistir registros válidos y exponer APIs REST para la operación y consulta. El objetivo es resolver problemas de datos inconsistentes, falta de trazabilidad y control centralizado, facilitando la toma de decisiones y la auditoría de resultados.

Endpoints principales
  • GET /api/routes/ Listar todas las rutas.
  • POST /api/routes/ Crear una nueva ruta.
  • POST /api/routes/import Importar rutas desde Excel.
  • POST /api/routes/execute Ejecutar rutas seleccionadas.
  • GET /api/execution-logs/ Consultar logs de ejecución.
